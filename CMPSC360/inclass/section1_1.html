IST 230, CMPSC 360: Section 1.1

  • 10.
    • a. The election is not decided.
    • b. Either the election is decided or the votes have been counted.
    • c. The election is not decided or the votes have been counted; OR: If the election is decided, then the votes have been counted.
    • d. If the votes have been counted, then the election is decided.
    • e. If the votes have not been counted, then the election is not decided.
    • f. If the election is not decided, then the votes have not been counted; OR: d.
    • g. The election is decided if and only if the votes have been counted.
    • h. Either the votes have not been counted, or the election is not decided and the votes have been counted.
  • 14.
    • a. r ∧ ¬q.
    • b. p ∧ q ∧ r.
    • c. r → p.
    • d. p ∧ ¬q ∧ r.
    • e. ( p ∧ q ) → r.
    • f. r ↔ ( p ∨ q ).
  • 18.
    • a. F → F = T.
    • b. F → F = T.
    • c. T → F = F.
    • d. T → T = T.
  • 22.
    • a. If you want to get promoted, then you wash the boss's car.
    • b. If the winds are from the south, then there is a spring thaw.
    • c. If you bought the computer less than a year ago, then the warranty is good.
    • d. If Willy cheats, then he gets caught.
    • e. If you can access the website, then you pay a subscription fee.
    • f. If you know the right people, then you get elected.
    • g. If Carol is on a boat, then Carol gets seasick.
