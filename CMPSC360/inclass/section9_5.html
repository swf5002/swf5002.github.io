  • 14. R = { (s1, s2) | s1 and s2 have the same nth letter (upper or lower case)}

    R is reflexive because for all s: (s,s) ∈ R.
    R is symmetric because for all s1, s2: if (s1,s2) ∈ R, then (s2,s1) ∈ R.
    R is transitive because for all s1, s2, s3: if (s1,s2) ∈ R, and s2,s3) ∈ R, then s1,s3) ∈ R.
    Therefore, R is an equivalence relation.
