  • 4.
    • a. P(18), P(19), P(20), and P(21)

      P(18) or 2⋅7 + 1⋅4 = 18
      P(19) or 1⋅7 + 3⋅4 = 19
      P(20) or 0⋅7 + 5⋅4 = 20
      P(21) or 3⋅7 + 0⋅4 = 21

      Note that we can get any other amount (greater than 17) merely by adding a 4-cent stamp to amount needed for the amount 4 cents less (which can be done in 4-cent and 7-cent stamps by inductive assumption):
      P(22) or 2 7-cent stamps + 4 4-cent stamps. i.e., add a 4-cent stamp to the 18-cent case.
      P(23) or 1 7-cent stamp + 4 4-cent stamps, i.e., add a 4-cent stamp to the 19-cent case.
      P(24) or 0 7-cent stamps + 6 4-cent stamps, i.e., add a 4-cent stamp to the 20-cent case.
      P(25) or 3 7-cent stamps + 1 4-cent stamp, i.e., add a 4-cent stamp to the 21-cent case.

    • b. P(k) for all k < n
    • c. P(n)
    • d.

      Assume: P(k) for all k < n
      Show: P(n). This is done by construction.

      Begin with k 4-cent stamps where n / 4 = k (with no remainder), and then consider the remainder n % k:
      Case 1: n % k = 0; use (n / 4) = k 4-cent stamps.
      Case 2: n % k = 1; use k-1 7-cent stamps and 1 4-cent stamp.
      Case 3: n % k = 2; use k-2 7-cent stamps and 4 4-cent stamps.
      Case 4: n % k = 3; use k-3 7-cent stamps and 6 4-cent stamps.
