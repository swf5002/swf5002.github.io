  • 4. Find f(2, f(3), f(4), and f(5), where f(0) = f(1) = 1, and for n = 2, 3, ....
    • a. f(n+1) = f(n) - f(n-1)

      f(2) = f(1) - f(0) = 1 - 1 = 0
      f(3) = f(2) - f(1) = 0 - 1 = -1
      f(4) = f(3) - f(2) = -1 - 0 = -1
      f(5) = f(4) - f(3) = -1 - -1 = 0

    • b. f(n+1) = f(n)f(n-1)

      f(2) = f(1)f(0) = 1(1) = 1
      f(3) = f(2)f(1) = 1(1) = 1
      f(4) = f(3)f(2) = 1(1) = 1
      f(5) = f(4)f(3) = 1(1) = 1

    • c. f(n+1) = (f(n))2(f(n-1)3

      f(2) = (f(1))2 + (f(0))3 = 1 + 1 = 2
      f(3) = (f(2))2 + (f(1))3 = 4 + 1 = 5
      f(4) = (f(3))2 + (f(2))3 = 25 + 8 = 33
      f(5) = (f(4))2 + (f(2))3 = 1089 + 125 = 1214

    • b. f(n+1) = f(n) / f(n-1)

      f(2) = f(1) / f(0) = 1 / 1 = 1
      f(3) = f(2) / f(1) = 1 / 1 = 1
      f(4) = f(3) / f(2) = 1 / 1 = 1
      f(5) = f(4) / f(3) = 1 / 1 = 1

  • 28a. S = {(a,b) | a ∈ Z+, b ∈ Z+, and a + b is even}

    BASE: let (1,1) ∈ S
    Then add (a,b) where a and b are both even OR both odd, but NOT where one is odd and the other is even.
    RECURSIVE:
    If (a,b) ∈ S, then (a+2,b) ∈ S
    If (a,b) ∈ S, then (a,b+2) ∈ S
    If (a,b) ∈ S, then (a+1,b+1) ∈ S

  • 48. Ackermann functions, A(m,n)
    • a. A(1,0) = 0, as m = 1 and n = 0
    • b. A(0,1) = 2⋅1 = 2, as m = 0
    • c. A(1,1) = 2, as m = 1 and n = 1
    • c. A(2,2) = A(1,A(2,1)) = A(1,2) = A(0,A(1,1)) = A(0,2) = 2 ⋅2 = 2
