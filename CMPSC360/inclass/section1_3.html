    You can do an 8-line truth table, or you can use an equivalence proof.

    Show: ( p → q ) ∧ ( p → r ) ≡ p → ( q ∧ r )

    1. ( p → q) ∧ ( p → r ) ≡            lefthand side
    2. ( ¬p ∨ q ) ∧ ( ¬p ∨ r ) ≡         2 applications of: p → q ≡ ¬p ∨ q
    3. ¬p ∨ ( q ∧ r ) ≡                       distribution of disjunction over conjunction
    4. p → ( q ∧ r )                           1 application of: p → q ≡ ¬p ∨ q; qed

    You can do an 8-line truth table, or you can use an equivalence proof.

    Show: ( p → q ) ∨ ( p → r ) ≡ p → ( q ∨ r )

    1. ( p → q) ∨ ( p → r ) ≡            lefthand side
    2. ( ¬p ∨ q ) ∨ ( ¬p ∨ r ) ≡         2 applications of: p → q ≡ ¬p ∨ q
    3. ( ¬p ∨ ¬p ) ∨ ( q ∨ r ) ≡         associativity and commutativity of disjunction
    4. ¬p ∨ ( q ∨ r ) ≡                       idempotence of disjunction
    5. p → ( q ∧ r )                           1 application of: p → q ≡ ¬p ∨ q; qed
  • 32.

    We show the following two propositions are not equivalent by finding an assignment of truth values that makes one true and the other false (so there is at least one row in which they have opposite truth values).

    1. ( p ∧ q ) → r
    2. ( p → r ) ∧ ( q → r )

    We can make 2 false by choosing q = T and r = F. This makes the right conjunct of 2 false, and hence makes the conjunction false. Now we try to make 1 true when q = T and r = F. This done by making p = F.

    Hence, the assignment of p = F, q = T, and r = F makes 1 true and 2 false; and it follows that 1 and 2 are not equivalent.
