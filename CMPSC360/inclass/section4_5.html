IST 230, CMPSC 360: Section 4.5

  • 2a. h(k) = k (mod 101)

    104,578,690 = k; h(k) = 104,578,690 (mod 101)
    104,578,690 / 101 = 1,035,432, and 104,578,690 % 101 = 58
    Therfore h(k) = 58.

  • 6. xn+1 = (4xn + 1) mod 7 and x0 = 3

    (4⋅3 + 1) mod 7 = 13 mod 7 = 6
    (4⋅6 + 1) mod 7 = 25 mod 7 = 4
    (4⋅4 + 1) mod 7 = 17 mod 7 = 3
    ... (sequence repeats indefinitely)

  • 18a. x11 = (x1 + x2 + ... + x10) mod 9

    7 + 5 + 5 + 5 + 6 + 1 + 8 + 8 + 7 +3 = 55
    55 = 6⋅9 + 1, so 55 mod 9 = 1.

  • 32a. d8 = (3d1 + 4d2 + 5d3 + 6d4 + 7d5 + 8d6 + 9d7) mod 11

    3 + 20 + 35 + 0 + 56 + 48 + 72 = 234, and 235 = 11⋅21 + 3
    And, 234 mod 11 = 3.
