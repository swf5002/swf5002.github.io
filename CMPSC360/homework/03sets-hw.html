SETS HOMEWORK

Total Points = 100

  1. (10 points) Use set builder notation to describe the following sets:. Let U = Z (all integers), O = odd integers, and E = even integers.

    1. { 1, 3, 5, 7 }
    2. { –4, –2, 0, 2 }
    3. { 1, 2, 3, 4, ... }
    4. { ..., –3, –2, 0, 1, 2, 3 }
    5. { ..., –3, –2, –1, 0, 1, 2, 3, .... }

  2. (10 points) Given the set S = { 0, 1, 2, 3 }.

    1. Determine the cardinality of the power set of S, show the formula used to do so.
    2. Show the power set of S: P ( S ).

  3. (10 points) Let A = { 0, 1, 2 }, B = { a, b } and C = { A, B }.

    1. Determine the cardinality of Cartesian product: A X B X C; show the formula used to do so.
    2. Show the set A X C X B.
    3. Show the set A X B X B.

  4. (10 points) Find the truth set of each of these predicates where the domain is as specified.

    1. P ( x ):   x2 = 2, domain = Z
    2. Q ( x ):   x2 = 2, domain = R
    3. R ( x ):   x2 > 4, domain = Z
    4. S ( x ):   x2 > 4, domain = R
    5. T ( x ):   x2 < x, domain = R

  5. (10 points) For each of the following, translate the proposition into English, and then state whether it is true or false and why.

    1. ∀x ∈ Z+ ∀y ∈ Z+ ( x − y ∈ Z+ )
    2. ∀x ∈ R ∃y ∈ R ( x ⋅ y = 1)
    3. ∀x ∈ Z ∃y ∈ Z ( x − y = 0)
    4. ∀x ∈ R ( 0x = 1 )
    5. ∃x ∈ R ∀y ∈ R ( x ⋅ y = x)

  6. (15 points) Let A, B, C, and D be sets. Using only set operation definitions and propositional identities:

    1. Prove ( A ∪ B ) − ( C ∪ D ) = ( ( A − C ) − D ) ∪ ( ( B − D ) − C)
    2. Find a counterexample to disprove ( A ∪ B ) − ( C ∩ D ) = ( A − C ) ∩ ( B − D )

  7. (10 points) Show by means of a membership table that both absorption laws are identities.

    3. Ai = [ 2, i ], that is, the closed interval of real numbers x between and i, inclusive of the end points

  9. (10 points) Given the following recursive definition of the successor function that generates a set-theoretic definition of the set of natural numbers, N: ∅ ∈ N, if n ∈ N, then { n } ∈ N.

    1. Generate a list of the first 5 elements of N.
    2. Give a definition of n + m, where n and m are natural numbers.
    3. Give a definition of n − m, where n and m are natural numbers.
