    1. procedure sum (n: integer)
{
   for i =: 1 to n do
      sum =: sum + 1
   return sum
}
     
    2. procedure increment ( )
{
   for i =: 1 to n do
      ai =: ai + 1
   return a
}
     
    3. procedure decrement (a: array of integers)
{
   for i =: 1 to n do
      ai =: ai − 1
}
     
    4. procedure divide (n: integer > 1 )
{
   i =: 1
   while n > 0 do
      n =: n − 1/i
      i =: 2 * 1
   return n
}

  3. (12 points) Use the following selection sort to sort 3, 4, 5, 6, 1, 2. (Warning: this selection sort is not quite the same as that in text p. 203.). Be sure to show EVERY step.
    4. procedure selection (a: array of integers a1...an, n: integer > 1 = length of list)
i = 1
while i < n do
   j =: i+1
   least =: i
   while j <= n do
      if a[j] < a[least]
         least = j
      j =: j+1
   if i ≠ least
      swap array position i with array position least
   i =: i + 1
return { array is in increasing order }

    Array: 3 4 5 6 1 2
