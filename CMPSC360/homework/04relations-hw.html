RELATIONS Homework

    Total Points = 170

  1. (6 points) Given the relation R = { ( a, b ) | a divides b } on the set { 1, 2, 3, 4, 5, 6 }.


    1. List all the ordered pairs in the relation.
    2. Display this relation graphically.
    3. Display this relation in tabular form.

  2. (12 points) Determine the following for the given relation R on the set { 1, 2, 3, 4, 5 } defined by the rule ( x, y ) ∈ R if and only if x = y – 1.


    1. List the elements of R.
    2. Find the domain of R.
    3. Find the range of R.
    4. List the elements of R-1.
    5. Find the domain of R-1.
    6. Find the range of R-1.

  3. (12 points) Determine whether the relation R on the set of all people is reflexive, symmetric, antisymmetric, and/or transitive where ( a, b ) ∈ R iff


    4. Relations: a. a is taller than b
    b. a and b were born on the same day
    c. a and b have a common grandparent

  4. (8 points) Let relation R = { ( 1, 2 ), ( 1, 3 ), ( 2, 3 ), ( 2, 4 ), ( 3, 1 ) } and relation S = { ( 2, 1 ), ( 3, 1 ), ( 3, 2 ), (4, 2 ) }.   Find S ∘ R.


  5. (12 points) Let relation R = { ( 1, 1 ), ( 1, 2 ), ( 1, 3 ), ( 2, 3 ), ( 2, 4 ), ( 3, 1 ), ( 3, 4 ), ( 3, 5 ), ( 4, 2 ), ( 4, 5 ), ( 5, 1 ), ( 5, 2 ), (5, 4 ) } on the set { 1, 2, 3, 4, 5 }.   Find R2 (i.e., R o R), R3, R4, R5.


  6. (12 points) Determine the matrix representation and digraphs for the following relations on { 1, 2, 3 }.


    7. a. { ( 1, 1 ), ( 1, 3 ), ( 2, 2 ), ( 3, 1 ), ( 3, 3 ) }
    b. { ( 1, 2 ), ( 2, 2 ), ( 3, 2 ) }
    c. { ( 1, 1 ), ( 1, 2 ), ( 1, 3 ), ( 2, 1 ), ( 2, 3 ), ( 3, 1 ), ( 3, 2 ), ( 3, 3 ) }

  7. (8 points) Given relation R = { ( a, a ), ( a, b ), ( b, c ), ( c, a ) }.   Determine the reflexive closure digraph from the digraph representing R.   Determine the symmetric closure digraph from the digraph representing R.


  8. (12 points) Given digraph
    determine which sequences of vertices form a path in this digraph.   Justify your answer if the sequence does not form a path.


    9. Sequences: a. a, b, c, d
    b. b, d, c, b, d
    c. a, a, b, d, e, d
    d. b, c, d, e, a, a, b
    e. b, c, c, b, d, e, d, e
    f. a, a, b, b, c, c, b, d, e

  9. (16 points) Prove that the relation R on the set of ordered pairs of positive integers such that ( ( a, b), ( c, d ) ) ∈ R iff ad = bc is an equivalence relation.


  10. (16 points) Determine if the relation R with the given digraph is an equivalence relation.

  11. (16 points) Determine if the relation R with the given digraph is a poset.

  12. (24 points) Consider the poset R = ( { 2, 4, 6, 9, 12, 18, 27, 36, 48, 60, 72 }, | ).


    1. Draw the Hasse diagram.
    2. Find the maximal elements.
    3. Find the minimal elements.
    4. Is there a greatest element?
    5. Is there a least element?
    6. Find all the upper bound of { 2, 9 }.
    7. Find the least upper bound of { 2, 9 }, if it exists.
    8. Find all the lower bounds of { 60, 72 }.
    9. Find the greatest lower bound of { 60, 72 }, if it exists.
    10. Find a compatible total order.

  13. (16 points) Determine whether these posets are lattices.


    1. ( { 1, 3, 6, 9, 12 }, | )
    2. ( { 1, 5, 25, 125 }, | )
    3. ( { Z }, ≥ )
    4. ( Þ( S ), ⊆ )
