FUNCTIONS Homework

    Total Points = 118

  1. (6 points) Determine whether f is a function from Z to R if


    1. f ( n ) = | n |
    2. f ( n ) = ( n4 + 1 )1/2
    3. f ( n ) = 1 / ( n2 – 6 )

  2. (10 points) Find the domain and range of these functions.


    1. the function that assigns to each pair of positive integers the first integer of the pair
    2. the function that assigns to each positive integer its largest decimal digit
    3. the function that assigns to a bit string the number of ones minus the number of zeros in the string
    4. the function that assigns to each positive integer the largest integer not exceeding the square of the integer
    5. the function that assigns to a bit string the length of longest string of consecutive ones in the string

  3. (16 points) Find these values.


    1. ⌊ 1.1 ⌋
    2. ⌈ 1.1 ⌉
    3. ⌊ –0.1 ⌋
    4. ⌈ –0.1 ⌉
    5. ⌈ 2.99 ⌉
    6. ⌈ –2.99 ⌉
    7. ⌊ ½ + ⌈ ½ ⌉ ⌋
    8. ⌈ ⌊ ½ ⌋ + ⌈ ½ ⌉ + ½ ⌉

  4. (16 points) Determine whether each of these functions from Z to Z is 1-1 or is an injection.   Justify your answer.


    1. f ( n ) = n – 1
    2. f ( n ) = n2 + 1
    3. f ( n ) = n3
    4. f ( n ) = ⌈ n / 2 ⌉

  5. (10 points) Determine whether f: Z x Z → Z is onto or is a surjection if


    1. f ( m, n ) = 2m – n
    2. f ( m, n ) = m2 – n2
    3. f ( m, n ) = m + n + 1
    4. f ( m, n ) = | m | – | n |
    5. f ( m, n ) = m2 – 4

  6. (16 points) Determine whether each of these functions is a bijection or 1-1 correspondence from R to R.


    1. f ( x ) = –3x + 4
    2. f ( x ) = –3x2 + 7
    3. f ( x ) = ( x + 1 ) / ( x + 2 )
    4. f ( x ) = x5 + 1

  7. (8 points) Let S = { –1, 0, 2, 4, 7 }.   let the range of f = Z. Find f ( S ) if


    1. f ( x ) = 1
    2. f ( x ) = 2x + 1
    3. f ( x ) = ⌈ x / 5 ⌉
    4. f ( x ) = ⌊ ( x2 + 1 ) / 3 ⌋

  8. (4 points) Find f ° g and g ° f where f ( x ) = x2 + 1 and g ( x ) = x + 2, are functions from R to R.


  9. (6 points) Decompose the function f ( x ) = 1 / 2x2 into simpler functions.


  10. (12 points) Draw the graph of the function f ( x ) = ⌈ x ⌉ + ⌊ x / 2 ⌋ from R to R.


  11. (16 points) Suppose that f is an invertible function from Y to Z and g is an invertible function from X to Y.   Show that the inverse of the composition f ° g is given by ( f ° g )-1 = g-1 ° f-1.


  12. (8 points) A function is 1-1 on the specified domain X.   By letting Y be the range of f, we obtain a bijection from X to Y.   Find the inverse function of f ( x ) = 6 + 27x–1, where X is set of real numbers.


  13. (12 points extra credit) Let x be a real number.   Show that ⌊ 3x ⌋ = ⌊ x ⌋ + ⌊ x + 1/3 ⌋ + ⌊ x + 2/3 ⌋
