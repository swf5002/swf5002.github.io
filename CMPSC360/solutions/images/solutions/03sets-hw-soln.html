  4. Find the truth set of each of these predicates where the domain is the set of integers.


    1. P ( x ):   x3 ≥ 1
    2. Q ( x ):   x2 = 2

  7. The symmetric difference of two sets A and B is the set A ⊕ B = ( A ∪ B ) – ( A ∩ B ).   Given a universe U, describe A ⊕ A, A ⊕ A′, U ⊕ A, and ∅ ⊕ A.


    8. SOLUTION: (18) A ⊕ A = ( A ∪ A ) – ( A ∩ A ) = A – A = ∅
    (19) A ⊕ A′ = ( A ∪ A′ ) – ( A ∩ A′ ) = U – ∅ = U
    (20) U ⊕ A = ( U ∪ A ) – ( U ∩ A ) = U – A = A′
    (21) ∅ ⊕ A = ( ∅ ∪ A ) – ( ∅ ∩ A ) = A – ∅ = A
