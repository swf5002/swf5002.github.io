FUNCTIONS


Introduction
Function
f from non-empty set A to non-empty set B, denoted by f : A → B and read as "f maps A to B", is an assignment of exactly one element of B to each element of A. The exactly one proviso means: for all a1 and a2 in A, if a1 = a2, then f(a1) = f(a2).  

The set A is said to be the domain of f and the set B is said to be the co-domain of f.

f ( a ) = b indicates that b is the unique element of B assigned by the function f to the element a of A.
The element b is said to be the image of a whereas the element a is said to be the pre-image of b.

The range of f is the set of all images of elements of A.

A function is sometimes also called a mapping or a transformation.

Equal Functions
f and g are equal if they have the same domain, have the same co-domain, and map elements of their common domain to the same elements of their common co-domain.
Sum of Functions
f1 : A → R and f2 : A → R, denoted by f1 + f2, is also a function from A to R defined by

( f1 + f2 ) ( x ) = f1 ( x ) + f2 ( x )
Product of Functions
f1 : A → R and f2 : A → R, denoted by f1 f2, is also a function from A to R defined by

( f1 f2 ) ( x ) = f1 ( x ) f2 ( x )
Subset Image
of subset S ⊆ A under the function f : A → B, denoted by f ( S ), consists of the images of the elements of S, i.e.

f ( S ) = { t | ∃s ∈ S ( t = f ( s ) ) }
or
f ( S ) = { f ( s ) | s ∈ S}

One-to-One and Onto Functions
Injection or One-to-One
A function f : A → B is 1–1 or one–to–one or injective iff for all a1 and a2 in A, f(a1) = f(a2) implies a1 = a2. This definition is often stated in the contrapositive as: a1 ≠ a2 implies f(a1) ≠ f(a2).

Strictly Increasing
A function f whose domain and co-domain are subsets of the set of real numbers is called increasing if f ( x ) ≤ f ( y ), and strictly increasing if f ( x ) < f ( y ), whenever x < y and x and y are in the domain of f.
Strictly Decreasing
A function f whose domain and co-domain are subsets of the set of real numbers is called decreasing if f ( x ) ≥ f ( y ), and strictly decreasing if f ( x ) > f ( y ), whenever x < y and x and y are in the domain of f.
Surjection or Onto
A function f : A → B is onto or surjective if ∀b ∈ B, ∃a ∈ A, f ( a ) = b.

Bijection or 1–1 Correspondence
A function f: A → B is 1–1 correspondence or bijective between A and B if f is both injective and surjective, that is, both one-to-one and onto.

Identity Function
on A is the function iA: A → A, where iA ( x ) = x, ∀x ∈ A, i.e. this function assigns each element to itself.   Observe that iA is 1-1 and onto, and is therefore a bijection.

Inverse Functions and Compositions of Functions
Inverse Function
of a bijective function f from the set A to the set B, denoted by f-1, is the function that assigns to an element b belonging to B the unique element a in A such that f ( a ) = b.   Hence, f-1 ( b ) = a when f ( a ) = b.

Invertible
function is a 1-1 Correspondence function, i.e., the function has an inverse.
Composition
of the function f and g, denoted by f ° g, is defined by
( f ° g ) ( a ) = f ( g ( a ) )


The Graphs of Functions
Graph
of a function f is the set of ordered pairs { ( a, b ) | a ∈ A ∧ f ( a ) = b }.

Some Important Functions
Floor Function
denoted by ⌊ x ⌋, assigns to a real number x the largest integer that is less than or equal to x.
Ceiling Function
denoted by ⌈ x ⌉, assigns to a real number x the smallest integer that is greater than or equal to x.
Greatest Integer Function
denoted by [ x ] is another name for the floor function.
USEFUL PROPERTIES
of the FLOOR FUNCTION
and the CEILING FUNCTION
where n is an integer
(1a) ⌊ x ⌋ = n iff n ≤ x < n + 1
(1b) ⌈ x ⌉ = n iff n – 1 < x ≤ n
(1c) ⌊ x ⌋ = n iff x – 1 < n ≤ x
(1d) ⌈ x ⌉ = n iff x ≤ n < x + 1
(2) x – 1 < ⌊ x ⌋ ≤ x ≤ ⌈ x ⌉ < x + 1
(3a) ⌊ – x ⌋ = – ⌈ x ⌉
(3b) ⌈ – x ⌉ = – ⌊ x ⌋
(4a) ⌊ x + n ⌋ = ⌊ x ⌋ + n
(4b) ⌈ x + n ⌉ = ⌈ x ⌉ + n
