Isomorphism of Graphs
Isomorphism
of two simple graphs G1 = ( V1, E1 ) and G2 = ( V2, E2 ) occurs when there is a 1-1 and onto function f: V1 → V2
with the property that a and b are adjacent in G1 iff f(a) and f(b) are adjacent in G2, for all a and b in V1.
OBSERVATION:
  • Since isomorphic graphs are 1-1 between the sets of vertices,
    then they must have the same number of vertices.
  • Since isomorphic graphs are 1-1 between the sets of vertices,
    then they must have the same number of edges.
  • The degrees of corresponding vertices in isomorphic simple graphs must be the same.

Another way of showing that two simple graphs are isomorphic is with an adjacency matrix.

CONNECTIVITY


Path
Path or Walk
of length n > 0 from vertex u to vertex v in an undirected graph G is a sequence of edges e1, e2, . . ., en of G
such that e1 = { x0, x1 }, e2 = { x1, x2 }, . . ., en = { xn-1, xn } where x0 = u and xn = v.
Circuit or Cycle or Closed Walk
is a path that begins and ends with the same vertex, i.e. u = v, and has length greater than 0.
Directed Path
of length n > 0 from vertex u to vertex v in an directed graph G is a sequence of edges e1, e2, . . ., en of G
such that e1 = ( x0, x1 ), e2 = ( x1, x2 ), . . ., en = ( xn-1, xn ) where x0 = u and xn = v.
Pass Through
A path or circuit is said to pass through its vertices.
Traverse
A path or circuit is said to traverse its edges.
Simple Path or Simple Circuit or Trail
is path or circuit or trail that does not contain an edge more than once.

Connectedness in Undirected Graphs
Connected
undirected graph is a graph with a path between every pair of distinct vertices of the graph.
THEOREM05: There is a simple path between every pair of distinct vertices of a connected undirected graph.

Connectedness in Directed Graphs
Strongly Connected
directed graph is a graph with a path from a to b and from b to a whenever a and b are vertices in a graph.
Weakly Connected
directed graph is a graph with a path between every two vertices in the underlying undirected graph.

EULER AND HAMILTONIAN PATHS


Euler Paths and Circuits
Euler Circuit
in a graph G is a simple circuit containing every edge of G.
Euler Path
in a graph G is a simple path containing every edge of G.
OBSERVATION: Necessary and Sufficient Conditions for Euler Circuits and Paths
Every vertex must have an even degree.

THEOREM07: A connected multigraph with at least two vertices has an Euler circuit iff each of its vertices has even degree.

ALGORITHM01: Construction of Euler Circuits
function Euler ( G: connected multigraph with all vertices of even degree )
    {
        circuit = a circuit in G beginning with an arbitrarily chosen vertex
                      with edges successively added to form a path that returns to this vertex ;
        H = G with the edges of this circuit removed ;

        while ( H has edges )
            {
                subcircuit = a circuit in H beginning at a vertex in H
                                    that also is an endpoint of an edge of circuit ;
                H = H with edges of subcircuit and all isolated vertices removed ;
                circuit = circuit with subcircuit inserted at the appropriate vertex ;
            }
    }

THEOREM09: Construction of Gray Codes
Let G1 denote the sequence 0, 1.   Define Gn in terms of Gn-1 by the following rules:
  1. Let Gn-1' denote the sequence obtained by prefixing each member of Gn-1 with 0.
  2. Let Gn-1" denote the sequence obatined by prefixing each member of Gn-1 with 1.
  3. Let Gn be the sequence consisting of Gn-1' followed by Gn-1".
Then Gn is a Gray code for every positive integer n.

THEOREM10: Dirac's Theorem
If G is a simple graph with n ≥ 3 vertices
such that the degree of every vertex in G is at least n / 2,
then G has a Hamiltonian circuit.

THEOREM11: Ore's Theorem
If G is a simple graph with n ≥ 3 vertices
such that deg ( u ) + deg ( v ) ≥ n for every pair of non-adjacent vertices u and v in G,
then G has a Hamiltonian circuit.

A Shortest-Path Algorithm
ALGORITHM02: Dijkstra's Algorithm
function Dijkstra ( G : weighted connected simple graph, with all weights positive )
    {
        {
            G has vertices a = v0, v1, . . ., vn = z and weights w ( vi, vj )
                where w ( vi, vj ) = ∞ if ( vi, vj ) is not an edge in G

            for ( i = 1; i <= n; i++ )
                {
                    L ( vi ) = ∞ ;
                }

            L ( a ) = 0 ;
            S = ∅ ;

            { the labels are now initialized so that the label of a is 0
                and all other labels are ∞, and S is the empty set }

            while ( z ∉ S )
                {
                    u = a vertex not in S with L ( u ) minimal ;
                    S = S ⋃ { u } ;

                    for ( all vertices v not in S )
                        {
                            if ( L ( u ) + w ( u, v ) < L ( v ) )
                                {
                                    L ( v ) = L ( u ) + w ( u, v ) ;
                                }

                            { this adds a vertex to S with minimal label
                                and updates the labels of vertices not in S }
                    }
            }
    }

Euler's Formula
THEOREM13: Euler's Formula
Let G be a connected planar simple graph with |V| = n vertices and |E| = k edges.  
The number of regions r in a planar representation of G is r = k – n + 2.

COROLLARY13-01: If G is a connected planar simple graph with |V| = n vertices and |E| = k edges,
where n ≥ 3, then k ≤ 3n – 6.

COROLLARY13-02: If G is a connected planar simple graph, then G has a vertex of degree not exceeding 5.

COROLLARY13-03: If G is a connected planar simple graph with |V| = n vertices and |E| = k edges,
where n ≥ 3 and has no circuits of length 3, then k ≤ 2n – 4.

Elementary Subdivision
is the operation of removing an edge { u, v } from a planar graph
