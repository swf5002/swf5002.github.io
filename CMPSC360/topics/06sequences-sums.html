SEQUENCES and SUMMATIONS


Introduction

Sequences are ordered lists of elements.   They are used to represent solutions to certain counting problems.   They are also an important data structure in computer science.   When the elements of an infinite set can be listed, the set is said to be countable.


Sequences

Arithmetic Progression
is a sequence of the form a, a + d, a + 2d, a + 3d, . . ., a = nd where the intial term a and the common difference d are real numbers.
EXAMPLE03: Determine the first six terms of an arithmetic progression with initial term 7 and common difference 5.
Recall that an arithmetic progression has the form a + i d where i = 0, 1, 2, . . ., n.
a + 0 * d = 7 + 0 * 5 = 7 + 0 = 7
a + 1 * d = 7 + 1 * 5 = 7 + 5 = 12
a + 2 * d = 7 + 2 * 5 = 7 + 10 = 17
a + 3 * d = 7 + 3 * 5 = 7 + 15 = 22
a + 4 * d = 7 + 4 * 5 = 7 + 20 = 27
a + 5 * d = 7 + 5 * 5 = 7 + 25 = 32
The first six terms of the arithmetic progression are 7, 12, 17, 22, 27, 32.
String

Length

Empty String

Cardinality
Cardinality of Sets
of set A and of set B are the same iff there is a one-to-one correspondence from A to B.
Countable
set is a set, S, that is finite (size = |S|) or has the same cardinality as the set of positive integers (|S| = ω).
Uncountable
set is a set that is not countable.
Aleph Null
denoted by ℵ0 is the cardinality of a S that is uncountable, i.e. |S| = ℵ0.
