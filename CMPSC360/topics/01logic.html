PROPOSITIONAL LOGIC


Introduction

The rules of logic give precise meaning to mathematical statements distinguishing between valid and invalid arguments.   Logic is the study of correct reasoning.   Logic's focus is the relationship between statements, and not the content of statements.

Propositions are the building blocks of logic.

Constructing New Logical Equivalences

The use of logical equivalences give rise to additional logical equivalences.

EXAMPLE09: Show that ¬ ( p ∨ ( ¬ p ∧ q ) ) and ¬ p ∧ ¬ q are logically equivalent
by developing a series of logical equivalences instead of using a truth table.
¬ ( p ∨ ( ¬ p ∧ q ) ) ≡ ¬ p ∧ ¬ ( ¬ p ∧ q ) by the first DeMorgan's Law
≡ ¬ p ∧ [ ¬ ( ¬ p ) ∨ ¬ q ] by the second DeMorgan's Law
≡ ¬ p ∧ ( p ∨ ¬ q ) by the Double Negation Law
≡ ( ¬ p ∧ p ) ∨ ( ¬ p ∧ ¬ q ) by the first Distributive Law
≡ F ∨ ( ¬ p ∧ ¬ q ) by the Negation Law for Conjunction
≡ ( ¬ p ∧ ¬ q ) ∨ F by the Commutative Law for Disjunction
≡ ¬ p ∧ ¬ q by the Identity Law for F

EXAMPLE10: Show that ( p ∧ q ) → ( p ∨ q ) is a tautology by developing
a series of logical equivalences instead of using a truth table.
( p ∧ q ) → ( p ∨ q ) ≡ ¬ ( p ∧ q ) ∨ ( p ∨ q ) by the LECS #1
≡ ( ¬ p ∨ ¬ q ) ∨ ( p ∨ q ) by the first DeMorgan's Law
≡ ¬ p ∨ ¬ q ∨ p ∨ q the above is an extended disjunction; remove parentheses
≡ ¬ p ∨ p ∨ ¬ q ∨ q by the Commutative Law for Disjunction
≡ ( ¬ p ∨ p ) ∨ ( ¬ q ∨ q ) by the Associative Law for Disjunction (add parentheses)
≡ T ∨ T by the Negation Law for Disjunction (used twice)
≡ T by the Domination Law for Disjunction

PREDICATES AND QUANTIFIERS


Introduction

Predicate Logic enables the expression of the meaning of a wide range of statements in ways that permit the reasoning and exploration of relationships between objects.


Predicates
Propositional Function
Let D be a set and variable x ∈ D.

P ( x ) is the proposition or the value of the propositional function P at x or with respect to D; another way of reading P ( x ) is: an x in domain D has the property P.

D is called the domain of the discourse of P.   The variable x in the propositional function P(x) is called a free variable.

R ( x1, x2, . . ., xn ) is the proposition or value of the propositional function R at the n-tuple ( x1, x2, . . ., xn ), and R is also called an n-place predicate or n-ary predicate; similarly it asserts of n elements in the domain D that they satisfy the relation R.

When the variables in a propositional function are assigned values, the resulting statement becomes a proposition with a certain truth value.

EXAMPLE11: Let P ( x ): x > 3
What are the truth values of P ( 7 ) and P ( 1 )?
P ( 7 ): 7 > 3 is T
P ( 1 ): 1 > 3 is F.
Let R ( x, y ): y = x + 3
What are the truth values of R ( 2, 4 ) and R ( 3, 6 ) ?
R ( 2, 4 ): 4 = 2 + 3 is F
R ( 3, 6 ): 6 = 3 + 3 is T

Predicates are also used for the verification of the correctness of a program, i.e. that the desired output, known as postcondition, is always produced given valid input, known as precondition.

Quantifiers
Quantification
expresses the extent to which a predicate is T over a range of elements.
Predicate Calculus
is the area of logic that deals with predicates and quantifiers.
Universal Quantification
of P ( x ) is the statement "P ( x ) for all values of x in the domain", is denoted by ∀x P ( x ).
∀ is called the universal quantifier and ∀x P ( x ) is read as "for all x P ( x )" or "for every x P ( x )".
An element, x, for which P ( x ) is F is called a counterexample of ∀x P ( x ).
Existential Quantification
of P ( x ) is the statement "there exists an element x in the domain such that P ( x )", is denoted by ∃x P ( x ).
∃ is called the existential quantifier and ∃x P ( x ) is read as "there is an x such that P ( x )" or "there is at least one x such that P ( x )" or "for some x P ( x )".

Other Quantifiers
Uniqueness Quantification
of P ( x ) is the statement "there exists a unique x in the domain such that P ( x )", denoted by ∃!x P ( x ) ( or ∃1x P ( x ) ).   ∃! is called the uniqueness quantifier.

The variable x in the proposition ∀x P ( x ), ∃x P ( x ), or ∃!x P ( x ) is called a bound variable, and this means x is bound by the quantifier. Propositions that have no free variables, that is, propositions in which all of the variables are bound, have a truth value of T or F.


Aristotelian Classification of English Sentences Using Quantifiers
A propositions have the form "All Ss are Ps" and are best symbolized with the universal quatifier and a conditional: ∀x( S ( x ) → P ( x )).
E propositions have the form "No Ss are Ps" and are best symbolized with the universal quantifier and a conditional with a negated predicate (P): ∀x( S( x ) → ¬ P( x )) OR as a negated existential quantifier of a conjunction: ¬∃x( S( x ) ∧ P( x )).
I propositions have the form "Some Ss are Ps" and are best symbolized with the existential quantifier and a conjunction: ∃x( S( x ) ∧ P( x )).
O propositions have the form "Some Ss are not Ps" and are best symbolized with the existential quatifier and a conjunction with a negated predicate (P): ∃x( S( x ) ∧ ¬ P( x )).

Quantifiers with Restricted Domains

The restriction of a universal quantification is the same as the universal quantification of a conditional statement. The statement: "all negative numbers have a square that is positive" can be symbolized as follows:

EXAMPLE13: ∀x < 0 ( x2 > 0 )

The restriction of an existential quantification is the same as the existential quantification of a conjunction. The statement: "There is a positive real number where square is 2" can be symbolized as:

EXAMPLE14: ∃x > 0 ( x2 = 2 )
∃x ( x > 0 ∧ x2 = 2 )

Binding Variables

When a quantifier is used on a variable, the occurrence of the variable is said to be bound.

An occurrence of a variable that is not bound by a quantifier is said to be free.

All variables that occur in a propositional function must be bound to turn it into a proposition which if T or F.


Logical Equivalences Involving Quantifiers
Logically Equivalent
Statements S and T with predicates and quantifiers are logically equivalent, denoted by S ≡ T, iff they have the same truth value no matter which predicates are substituted into these statements and which domain of discourse is used for the variables in these propositional functions.
EXAMPLE16: Prove that ∀x ( P ( x ) ∧ Q ( x ) ) and ∀x P ( x ) ∧ ∀x Q ( x ) are logically equivalent.
If proven, this indicates that ∀ can be distributed over conjunction.
First, show that if ∀x ( P ( x ) ∧ Q ( x ) ) is T, then ∀x P ( x ) ∧ ∀x Q ( x ) is T.
Second, show that if ∀x P ( x ) ∧ ∀x Q ( x ) is T, then ∀x ( P ( x ) ∧ Q ( x ) ) is T.
Suppose ∀x ( P ( x ) ∧ Q ( x ) ) is T.
This means that for a ∈ D, P ( a ) ∧ Q ( a ) is T.
Since P ( a ) is T and Q ( a ) is T for every element in D, then ∀x P ( x ) and ∀x Q ( x ) are both T.
Hence, ∀x P ( x ) ∧ ∀x Q ( x ) is T.
Suppose ∀x P ( x ) ∧ ∀x Q ( x ) is T.
It follows that ∀x P ( x ) is T and ∀x Q ( x ) is T.
Hence, if a ∈ D, then P ( a ) is T and Q ( a ) is T.
It follows that for all a, P ( a ) ∧ Q ( a ) is T.
It follows that ∀x ( P ( x ) ∧ Q ( x ) ) is T
\ ∀x ( P ( x ) ∧ Q ( x ) ) ≡ ∀x P ( x ) ∧ ∀x Q ( x ).

When the domain of discourse if finite, that is, the domain consists of a1, a2, ..., an, then a universally quantified proposition, ∀x P( x ), is equivalent to: P( a1 ) ∧ P( a2 ) ∧ ... ∧ P( an ). When the domain of discourse if finite, that is, the domain consists of a1, a2, ..., an, then an existentially quantified propositions, ∃x P( x ), is equivalent to: P( a1 ) ∨ P( a2 ) ∨ ... ∨ P( an ).

EXAMPLE19: Let P ( x, y ): x + y = y + x, where the domain of the variables is the set of real numbers.
What are the truth values of ∀x ∀y P ( x, y ) and ∀y ∀x P ( x, y )?
∀x ∀y P ( x, y ) denotes "for all real numbers x, for all real numbers y, x + y = y + x".
By the commutative law for addition, x + y = y + x
Therefore, ∀x ∀y P ( x, y ) is T.
∀y ∀x P ( x, y ) denotes "for all real numbers y, for all real numbers x, x + y = y + x".
And again, ∀y ∀x P ( x, y ) is T.
This example confirms that the order of nested universal quantifiers only are interchangeable.
Let Q ( x, y ): x + y = 0, where the domain of the variables is the set of real numbers.
What are the truth values of ∃x ∀y Q ( x, y ) and ∀y ∃x Q ( x, y )?
∃x ∀y Q ( x, y ) denotes "there is a real number x such that for every real number y, x + y = 0".
No matter what value of x is chosen, there is only one value of y for which x + y = 0.
There is no real number x such that x + y = 0 for all real numbers y.
Hence, ∃x ∀y Q ( x, y ) is F.
∀y ∃x Q ( x, y ) denotes "for all real numbers y, there is a real number x, such that x + y = 0".
For a given real value y, there is a real number x such that x + y = 0, i.e. y = – x.
Hence, ∀x ∃y Q ( x, y ) is T.
This example illustrates that the order in which quantifiers appear makes a difference.
Observe that if ∃x ∀y Q ( x, y ) is T, then ∀y ∃x Q ( x, y ) is also T.
However, if ∀y ∃x Q ( x, y ) is T, it is does not necessarily follow that ∃x ∀y Q ( x, y ) is T.
Let R ( x, y, z ): x + y = z, where the domain of all the variables is the set of real numbers.
What are the truth values of ∀x ∀y ∃z R ( x, y, z ) and ∃z ∀x ∀y R ( x, y, z )?
∀x ∀y ∃z R ( x, y, z ) denotes "for all real numbers x and all real numbers y there is a real number z such that x + y = z".
For a given real number x and a given real number y, there is indeed a real number z for which x + y = z.
Hence, ∀x ∀y ∃z R ( x, y, z ) is T.
∃z ∀x ∀y R ( x, y, z ) denotes "there is a real number z such that for all real numbers x and all real numbers y, x + y = z".
There is no real number z for which x + y = z for all real numbers x and all real numbers y.
Hence, ∃z ∀x ∀y R ( x, y, z ) is F.

Translating Mathematical Statements into Statements Involving Nested Quantifiers
EXAMPLE20: Translate the statement "The sum of two positive integers is always positive".
Rewrite the statement: "For every two integers, if these integers are both positive, then the sum of these integers is positive".
Let the domain be all integers.
"For all x and y, if x > 0 and y > 0, then x + y > 0".
∀x ∀y (x > 0 ∧ y > 0) →M/font> x + y > 0
Translate the statement "Every real number except zero has a multiplicative inverse".
Rewrite the statement "for every real number x except 0, x has a multiplicative inverse".
"For every real number x, if x ≠ 0, then there exists a real number y such that xy = 1".
Let the domain be all real numbers.
∀x ( ( x ≠ 0 ) → ∃y ( xy = 1 ) )

Translating from Nested Quantifiers into English
EXAMPLE21: Translate the statement ∀x ( C ( x ) ∨ ∃y ( C ( y ) ∧ F ( x, y ) ) ) into English where
C ( x ): x has a computer
F ( x, y ): x and y are friends; note that F is a symmetric relation, that is F ( x, y ) ↔ F ( y, x ),
and the domain is the students in a school.
"For every student x, x has a computer or there is a student y that that has a computer and is a friend of x".
"Every student has a computer or a friend with a computer".
Translate the statement ∃x ∀y ∀z ( ( F ( x, y ) ∧ F ( x, z ) ∧ y ≠ z) → ¬ F(y, z) ) into English where
F ( a, b ): a and b are friends
and the domain is the students in a school.
( ( F ( x, y ) ∧ F ( x, z ) ∧ ( y ≠ z ) ) → ¬ F ( y, z ) ) states that "there is a x such that for every y and z: if x and y are friends, and x and z are friends, and y is not z, then y and z are not friends".

Translating English Sentences into Logical Expressions
EXAMPLE22: Let L ( x, y ) be the statement "x loves y"
where the universe of discourse for both x and y is the set of all people in the world. Sadly, L is not a symmetric relation, that is L ( x, y) does not imply L ( y, x).
  Express each of the following sentences in terms of L ( x, y ), quantifiers, and logical connectives.
Note that the first argument in the propositional function is the lover and the second argument is the lovee.
• Everybody loves Jerry.
∀x L ( x, Jerry )
• Everybody loves somebody.
∀x ∃y L ( x, y )
Observe that somebody being loved depends on the person doing the loving, so the univeral quantifier has to come first.
• There is somebody whom everybody loves.
∃y ∀x L ( x, y )
Observe that one lovee works for all lovers, so the existential quantifier has to come first.
• Nobody loves everybody.
( ¬ ∃x ∀y L ( x, y ) ) ≡ ( ∀x ¬ ∀y L ( x, y ) ) ≡ ( ∀x ∃y ¬ L ( x, y ) )
• There is somebody whom Lydia does not love.
∃y ¬ L ( Lydia, y )
• There is somebody whom no one loves.
∃y ∀x ¬ L ( x, y )
• There is exactly one person whom everybody loves.
∃!y ∀x L(x, y)
Note that for the existence of a unique object satisfying a certain condition, the notation includes the use an exclamation point.
This is also equivalent to the following:
∃y ∀x ( L ( x, y ) ∧ ∀z ( L ( x, z ) → z = y )
• There are exactly two people whom Lynn loves.
∃x ∃y ( x ≠ y ∧ L ( Lynn, x ) ∧ L ( Lynn, y ) ∧ ∀z ( L ( Lynn, z ) → ( z = x ∨ z = y ) )
• Everyone loves himself or herself.
∀x L ( x, x )
• There is someone who loves no one besides himself or herself.
∃x ( L ( x, x ) ∧ ∀y ( L ( x, y ) → x = y ))
This is asserting that everyone who is loved by the lover is in fact the lover himself or herself.

Negating Nested Quantifiers
EXAMPLE23: Use quantifiers to express the statement that "There does not exist a woman who has taken a flight on every airline in the world."
Let P ( w, f ) denote w had taken flight f.
Let Q ( f, a ) denote f is a flight on airline a.
P ( w, f ) ∧ Q ( f, a ) denotes w had taken flight f and f is a flight in airline a.
∃w ∀a ∃f ( P ( w, f ) ∧ Q ( f, a ) ) denotes "there exists a woman w who has taken a flight f on every airline a in the world".
¬ ∃w ∀a ∃f ( P ( w, f ) ∧ Q ( f, a ) ) denote "there does not exist a woman w who had taken a flight f on every airline a in the world".
Successively apply DeMorgan's Laws for Quantifiers.
¬ ∃w ∀a ∃f ( P ( w, f ) ∧ Q ( f, a ) ) ≡ ∀w ¬ ∀a ∃f ( P ( w, f ) ∧ Q ( f, a ) )
≡ ∀w ∃a ¬ ∃f ( P ( w, f ) ∧ Q ( f, a ) )
≡ ∀w ∃a ∀f ¬ ( P ( w, f ) ∧ Q ( f, a ) )
≡ ∀w ∃a ∀f ( ¬ P ( w, f ) ∨ ¬ Q ( f, a ) )

Equivalence Relations
Reflexive Relation
A reflexive relation is a relation R that is true of everything in a domain with respect to itself; that is: ∀x R ( x, x). In mathematics the "equals" relation is reflexive, because everthing equals itself, x = x. On the other hand, the "loves" relation in the people domain is not necessarily reflexive, because not everybody loves themselves.

Symmetric Relation
A symmetric relation is a relation R for which R ( x, y) implies R ( y, x) for every x and y in a domain; that is: ∀x ∀y ( R ( x, y) → R ( y, x) ). In mathematics the "equals" relation is symmetric, because if x = y then y = x.

Transitive Relation
A transitive relation is a relation R for which R ( x, y) and R ( y, z ) implies R ( x, z) for every x, y and z in a domain; that is: ∀x ∀y ∀z ( ( R ( x, y) ∧ R ( y, z ) ) → R ( x, z) ). In mathematics the "equals" relation is transitive, because if x = y and y = z then x = z.

Equivalence Relation
An equivalence relation is a relation that is relexive, symmetric, and transitive (this is also covered again later on). In mathematics the "equals" relation is an equivalence relation; in geometry the "is congruent to" relation is an equivalence relation.
