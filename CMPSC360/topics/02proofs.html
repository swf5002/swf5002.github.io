RULES OF INFERENCE


Introduction
Proofs
are valid arguments that establish the truth of mathematical statements.
Rules of Inference
are the basic tools for deducing new statements from existing statements for establishing the truth of statements in constructing proofs.

Fallacies are common forms of incorrect reasoning which lead to invalid arguments.


Valid Arguments in Propositional Logic

NOTATION:   ∴ denotes "therefore".

NOTATION:   |− denotes "logically implies". A proposition p logically implies a proposition q iff it is impossible for p to be true and q false.

Argument
in propositional logic is a sequence of particular propositions (in English) where all but the final proposition are called premises and the final proposition is called the conclusion.
Valid Argument
An argument is valid if the truth of all its premises logically implies that the conclusion is true. That is, it is impossible for all the premises to be true and the conclusion false.
Argument Form
in propositional logic is a sequence of simple or compound propositions involving propositional variables. That is, an argument form is an English language argument symbolized in propositional logic.
Valid Argument Form
If regardless of the uniform substitutions of particular propositions for propositional variables in an argument form's premises and conclusion: if all the premises are T, then the conclusion remains T, then the argument form is valid. That is, no English language argument with the argument's form has all true premises and a false conclusion.

From the definition of a valid argument form, it follows that the argument form with premises p1, p2, . . ., pn and conclusion q is valid, iff ( p1 ∧ p2 ∧ . . . ∧ pn ) → q is a tautology. In propositional logic we show an argument form is invalid by showing that is it possible for all the form's premises to be true and the form's conclusion false. That is, an argument form is invalid if there is at least assignment of truth values to the propositional variables where the premises are all true and the conclusion is false. Alternatively, we can show an argument form is invalid by producing a counterexample, which is an English language argument with the argument's form that has all true premises and a false conclusion.

All of the rules of inference are valid argument forms, where it is not possible for the form's premises to be true and the form's conclusion false. In a nutshell, valid rules of inference never lead from truths to a falsity.


Rules of Inference for Propositional Logic

In each rule of inference note that in all cases in which ALL the premises are true the conclusion is also true. The rules of inference are stated in terms of the Greek letters: α (alpha), β (beta), γ (gamma), and δ (delta). These Greek letters range over propositional formulas. Propositional formulas of any complexity can be uniformly substituted for the Greek letters and the resulting rule of inference is a valid argument form. As such the rules of inference never lead from true premises to a false conclusion.

Resolution

Many of the computer programs that have been developed to automate the task of reasoning and proving theorem make use of the rule of inference known as resolution which is based on the tautology [ ( p ∨ q ) ∧ ( ¬ p ∨ r ) ] → ( q ∨ r ).   The final disjunction ( q ∨ r ) is known as the resolvent.

To construct proofs in propositional logic using resolution as the ONLY rule inference, the hypotheses and conclusion must be expressed as clauses, which are disjunctions of variables and negations of these variables.

The following proof makes use of the Resolution rule of inference.

In what follows a synonym of "premise" is "hypothesis".

EXAMPLE01: Given: Hypotheses: ( p ∧ q ) ∨ r
r → s
Conclusion: p ∨ s
Prove: The hypotheses imply the conclusion.
Proof: (1) ( p ∧ q ) ∨ r the first hypothesis
(2) r ∨ ( p ∧ q ) by the Commutative Law for Disjunction
(3) ( r ∨ p ) ∧ ( r ∨ q ) by the Distributive Law for Disjunction over Conjunction
(4) r ∨ p by Simplification from (3)
(5) r → s the second hypothesis
(6) ¬ r ∨ s by the LECS #1 from (5)
(7) p ∨ s from (4) and (6) by Resolution which is the conclusion

Rules of Inference for Quantified Statements

In the following rules the Greek letters φ (phi), χ (chi), ψ (psi), and θ (theta), are meant to range over well-formed predicates of any complexity.

RULES OF INFERENCE for QUANTIFIED STATEMENTS (RIQS)
NAME RULE OF INFERENCE
Universal Instantiation (UI)
∀x φ ( x ) , x ∈ D (unrestricted)
∴ φ ( c ) , c ∈ D
Universal Generalization (UG)
φ ( c ) , for any c ∈ D (restricted; c must be a constant new to the proof)
∴ ∀x φ ( x ) , x ∈ D
Existential Instantiation (EI)
∃x φ ( x ) , x ∈ D (restricted to constants new to the proof)
∴ φ ( c ) , for some c ∈ D
Existential Generalization (EG)
φ ( c ) , for some c ∈ D (unrestricted)
∴ ∃x φ ( x ) , x ∈ D

The restricted rules of inference must be used very carefully.

The Universal Instantiation rule is unrestricted: from a universally quantified proposition you can instantiate it to any c in the domain; that is, from "for all x φ of x" you can deduce "φ of c" for ANY c in the domain.

The Existential Generalization rule is unrestricted: from a propositional that asserts an element of the domain, c, has the property φ, you can deduce that "there is an x with the property φ."

The Existential Instantiation rule is restricted. The restriction requires that the c (in the domain) that you choose have the property φ must be new to the proof, that is it appears nowhere in the proof previously.

Similarly, the Universal Generalization rule is restricted; in order to prove a universal proposition, you must have shown that φ is a property of c, where c is ANY element in the domain; that is the derivation of φ ( c ) must be in a sub-proof in which the constant c is a constant new to the proof.

The use of these restrictions is illustrated in the examples below.


Using Rules of Inference to Build Arguments
EXAMPLE03: Given: Hypotheses: Everyone in Discrete Mathematics loves proofs.
Some in Discrete Mathematics have never taken calculus.
Conclusion: Someone who loves proofs has never taken calculus.
Prove: The conclusion follows from the hypotheses.
Proof: Let D ( x ) denote x is in Discrete Mathematics.
Let P ( x ) denote x loves proofs.
Let C ( x ) denote x has taken calculus.
The set of discourse is the set of students.
(1) The first hypothesis is ∀x ( D ( x ) → P ( x ) )
(2) The second hypothesis is ∃x ( D ( x ) ∧ ¬ C ( x ) )
(3) D ( c ) ∧ ¬ C ( c ) by Existential Instantiation; we MUST do this first as the constant c must be new to the proof
(4) D ( c ) by Simplification and (3)
(5) ¬ C ( c ) by Simplification and (3)
(6) D ( c ) → P ( c ) by Universal Instantiation and (1); this rule is unrestricted
(7) P ( c ) by Modus Ponens and (4) and (6)
(8) P ( c ) ∧ ¬ C ( c ) by Conjunction and (5) and (7)
(9) ∃x ( P ( x ) ∧ ¬ C ( x ) ) by Existential Generalization which is the conclusion

The above is a formal proof. In what follows we will be utilizing a more informal presentation so as not to become bogged down in minutiae.


Combining Rules of Inference for Propositions and Quantified Statements
EXAMPLE04: Given: Hypotheses: Everyone likes either Corel Office or Microsoft Office.
Karen does not like Microsoft Office.
Conclusion: Karen likes Corel Office.
Prove: Show that the conclusion follows from the hypotheses.
Proof: Let C ( x ) denote x likes Corel Office.
Let M ( x ) denote x likes Microsoft Office.
The set of discourse is the set of all people.
(1) The first hypothesis is ∀x ( C ( x ) ∨ M ( x ) )
(2) The second hypothesis is ¬ M ( Karen )
(3) By Universal Instantation of (1), we have C ( Karen ) ∨ M ( Karen )
(4) By Disjunctive Syllogism, and (2) and (3) we have C ( Karen ), which is the conclusion

Understanding How Theorems Are Stated

Many theorems assert that a certain property holds for all elements in a domain.   The precise statement of such a theorem should include a universal quantifier.   However, the convention is to omit it and assume that it is understood.

EXAMPLE05: The statement If x > y, where x and y are positive integers, then x2 > y2.
really means For all positive integers x and y, if x > y, then x2 > y2.

In this case we would begin by making the domain of discourse all positive real numbers. Next on the assumption that x > y, where x and y are ANY elements of the domain, we prove that x2 > y2.

Direct Proofs
Direct Proof
is based on proving that p → q is T.   Therefore, with direct proof, it is assumed that p is T and then using other axioms, definitions, and previous derived theorems, shows that q is T.
Even Integer (Definition)
The integer n is even iff there exists an integer k, k ≠ 0, such that n = 2k.
Odd Integer (Definition)
The integer n is odd iff there exists an integer k such tha n = 2k + 1 or n = 2k – 1.

A known fact about integers is: all integers (except 0) are either even or odd (note that 0 is neither even nor odd).

EXAMPLE06: Given: Hypothesis p: n is odd, where n is an integer
Conclusion q: n2 is odd.
Prove: p → q is T
Proof: Assume that n is odd, which is the hypothesis.
|− n = 2k + 1 where k is an integer.
|− n2 = ( 2k + 1 )2
|− n2 = 4k2 + 4k + 1
|− n2 = ( 4k2 + 4k ) + 1
|− n2 = 2 ( 2k2 + 2k ) + 1
|− Let c = 2k2 + 2k
|− n2 = 2c + 1
Hence, n2 is odd, which is the conclusion.
∴ p → q is T

EXAMPLE07: Given: Hypothesis p: m and n are both perfect squares, where m and n are integers
Conclusion q: mn is also a perfect square.
Prove: p → q is T
Proof: Assume that m and n are both perfect squares, which is the hypothesis
m = s2, where s is an integer and
n = t2, where t is an integer
|− mn = s2 t2
|− mn = ( st )2
Hence, mn is a perfect square, which is the conclusion.
∴ p → q is T

Other Proofs
Proof by Contraposition
is based on the logical equivalence that p → q ≡ ¬ q → ¬ p.   Hence, proof by contraposition establishes p → q is T by proving that ¬ q → ¬ p is T.
EXAMPLE08: Given: Hypothesis p: 3n + 2 is odd, where n is an integer
Conclusion q: n is odd
Prove: p → q is T by proof by contraposition.
Proof: Assume that n is even, which is the negation of the conclusion.
|− n = 2c, for some integer c
|− 3n + 2 = 3 ( 2c ) + 2
|− 3n + 2 = 6c + 2
|− 3n + 2 = 2 ( 3c + 1 )
|− 3n + 2 is even, which is the negation of the hypothesis.
Hence, p → q is proved by using p → q ≡ ¬ q → ¬ p.
Proof by Contradiction
shows p → q by assuing p is T and q is F, and then deducing a contradiction.
EXAMPLE09: Given: Hypothesis p: n = ab, where n, a and b are positive integers
Conclusion q: a ≤ n½ or b ≤ n½
Prove: p → q is T
Proof: Assume that a ≤ n½ or b ≤ n½ is F, which is the negation of the conclusion, that is the negation of a disjunction, which by DeMorgan's Laws is equivalent to the conjunction of the negations.
|− a > n½ and b > n½, now multiply the left sides of each inequality and the right sides of each inequality
|− ab > (n½)(n½) = n1 = n, a contradiction of the hypothesis
Hence, p → q is T.
Vacuous Proof
is proving p → q is T by proving that p is F. This is a strange proposition, but it is true because the antecedent of p → q is F.
EXAMPLE10: Given: Hypothesis p: 0 = 1
Conclusion q: ∀x ( x ≠ x )
Prove: p → q is T by vacuous proof
Proof: Assume p: 0 = 1, which is F. Hence, any implication with this antecedent is T: ∴ p → q is T.
Note: q was not needed in the proof
Trivial Proof
is proving p → q is T by proving that q is T.
EXAMPLE11: Given: Hypothesis p: 1 = 1
Conclusion q: ∀x ( x = x )
Prove: p → q is T by trivial proof
Proof: The consequent is always T. Hence, any implication with this consequent is T: ∴ p → q is T.
Note: p was not needed in the proof
Proof Strategy
When proving a statement of the form ∀x ( P ( x ) → Q ( x ) ), first attempt to use direct proof.   If direct proof is not going anywhere, then try proof by contraposition.
Rational Number
is a real number r that can be expressed as a fraction p / q where q ≠ 0 and p and q are both integers.
Irrational Number
is a real number that is not rational.
EXAMPLE12: Given: Hypothesis p: r and s are both rational numbers, where r and s are integers
Conclusion q: r + s is a rational number
Prove: The sum of two rational numbers is rational
Proof: First, try direct proof.
Suppose r and s are both rational numbers.
|− by definition, r = t / u where u ≠ 0
|− s = v / w where w ≠ 0.
|− r + s = t / u + v / w = ( tw + uv ) / ( uw ), where uw ≠ 0
Hence, r + s is a rational number.
Direct proof succeeded.

EXAMPLE13: Given: Hypothesis p: n2 is odd, where n is an integer
Conclusion q: n is odd
Prove: p → q is T
Proof: First, try direct proof.
Assume n2 is odd.
|− n2 = 2k + 1 for some integer k
|− n = ( 2k + 1 )½ which does not prove that n is odd.
Hence, direct proof failed.
Next, try proof by contraposition.
Assume n is even
|− n = 2k for some integer k
|− n2 = ( 2k )2 = 4k2 = 2 ( 2k2 )
∴ n2 is even.
Proof by contraposition succeeded.

Proofs by Contradiction
Proof by Contradiction
is based on the logical equivalence that p → q ≡ (( p ∧ ¬ q ) → ( r ∧ ¬ r ) ).   Hence, proof by contradiction establishes p → q is T by assuming that the hypothesis p is T and that the conclusion q is F and by using p and ¬ q and other axioms, definitions, and previously derived theorems, derives a contradiction, i.e. a proposition of the form r ∧ ¬ r.
EXAMPLE14: Prove: 2½ is irrational.
Proof: Let p: 2½ is irrational.
Then ¬ p: 2½ is rational. Assume ¬ p.
|− there exists integers a and b such that 2½ = a / b
where a and b have no common factors so that a / b is reduced to lowest terms.
|− ( 2½ )2 = ( a / b )2
|− 2 = a2 / b2
|− 2b2 = a2
|− a2 is even
|− a is also even
|− a = 2c for some integer c
|− 2b2 = a2 = ( 2c )2
|− 2b2 = 4c2
|− b2 = 2c2
|− b2 is even
|− b is also even
|− 2 is a common factor of a and b (contradicting reduction to lowest terms)
|− ¬ p is F and p is T

EXAMPLE15: Given: Hypothesis p: 3n + 2 is odd, where n is an integer
Conclusion q: n is odd
Prove: p → q is T by proof by contradiction.
Proof: To give a proof by contradiction, assume that both p and ¬ q are T.
|− ¬ q: n is even
|− Assume 3n + 2 is odd, which is the hypothesis
|− and n is even, which is the negation of the conclusion.
|− n = 2k for some integer k
|− 3n + 2 = 3 ( 2k ) + 2 = 6k + 2 = 2 ( 3k + 1 )
|− Let c = 3k + 1
|− 3n + 2 = 2c
|− ¬ p: 3n + 2 is even
p (the hypothesis) was assumed to be T and ¬ p is proven to be T, which is a contradiction
∴ p → q is T
Proof of Equivalence
is proving a theorem that is a biconditional statement, i.e a statement of the form p ↔ q, by showing that p → q and q → p are both T.   This is based on the tautology ( p ↔ q ) ↔ [ ( p → q ) ∧ ( q → p ) ].

Proof: Let p: n is odd
Let q: n2 is odd
Theorem: p ↔ q
p → q was proven in EXAMPLE06
q → p was proven in EXAMPLE13

(Dis)Proof by Counterexample
A statement of the form ∀x P(x) can be shown to be F by finding one counterexample, i.e., find an example x for which P(x) is F.
EXMPLE18: Prove: Every positive integer is the sum of the squares of two integers.
Proof: Consider the integer 3.
|− The only positive integers less than 3 are 0, 1, 2.
|− Of these integers, only the squares 02 = 0 and 12 = 1 are less then 3.
|− 3 ≠ 02 + 12
Hence, the statement is F.

Mistakes in Proofs
EXAMPLE19: Prove: If a = b, where a > 0, b > 0, then 1 = 2
Proof: Assume a = b, where a > 0, b > 0
Step Reason
(1) a = b Given
(2) a2 = ab Multiply both sides of (1) by a
(3) a2 – b2 = ab – b2 Subtract b2 from both sides of (2)
(4) ( a + b ) ( a – b ) = b ( a – b ) Factor both sides of (3)
(5) a + b = b Divide both sides of (4) by a – b; note: a – b = 0, and division by 0 is undefined!
(6) b + b = b Replace a with b
(7) 2b = b Simplify (6)
(8) 2 = 1 Divide both sides of (7) by b

PROOF METHODS AND STRATEGY


Exhaustive Proof and Proof by Cases
Proof by Cases
is proving a conditional statement of the form ( p1 ∨ p2 ∨ ... ∨ pn ) → q based on the tautology

[ ( p1 ∨ p2 ∨ ... ∨ pn ) → q ] ↔ [ ( p1 → q ) ∧ ( p2 → q ) ∧ ... ∧ ( ( pn → q ) ) ]

Proof by cases must cover all possible cases that arise in a theorem.
Exhaustive Proof
is a special type of proof by cases where each case involves checking a single example.
Perfect Power
is an integer that equals na, where a is an integer greater than 1.
EXAMPLE21: Prove: Prove that the only consecutive positive integers not exceeding 100 that are perfect powers are 8 and 9.
Proof: Consider the square of positive integers not exceeding 100.
|− 12 = 1, 22 = 4, 32 = 9, 42 = 16, 52 = 25, 62 = 36, 72= 49, 82 = 64, 92 = 81, 102 = 100
Consider the cubes of positive integers not exceeding 100.
|− 13 = 1, 23 = 8, 33 = 27, 43 = 64
Consider the fourth powers of positive numbers not exceeding 100.
|− 14 = 1, 24 = 16, 34 = 81.
Consider the fifth powers of positive numbers not exceeding 100.
|− 15 = 1, 25 = 32.
Consider the sixth powers of positive numbers not exceeding 100.
|− 16 = 1, 26 = 64.
There are no powers of positive integers higher than the sixth power not exceeding 100, other than 1.
Listing all the numbers together in ascending order, we obtain 1, 4, 8, 9, 16, 25, 27, 32, 36, 49, 64, 81, 100.
Note that the only consecutive numbers on the list of perfect powers are 8 and 9.

EXAMPLE22: Prove: if n is an integer, then n2 ≥ n
Proof: There are three cases to consider for every n, either n < 0, n = 0, or n > 0.
Consider n < 0.
|− n2 > 0.
Consider n = 0.
|− n2 = 02 = 0.
Consider n > 0.
|− n2 > 0.
∴ n2 ≥ n.
Absolute Value
of n, denoted by |n|, equals to n when n ≥ 0 and equals – n when n < 0.
EXAMPLE23: Prove: | xy | = | x | | y |, where x and y are real numbers.
Proof: There are four cases to consider, when x ≥ 0 and y ≥ 0, when x ≥ 0 and y < 0, when x < 0 and y ≥ 0, and when x < 0 and y < 0.
Consider x ≥ 0 and y ≥ 0.
|− | xy | = xy = | x | | y |
Consider x ≥ 0 and y < 0.
|− | xy | = x ( –y ) = | x | | y |
Consider x < 0 and y ≥ 0.
|− | xy | = ( –x ) y = | x | | y |
Consider x < 0 and y < 0.
|− | xy | = ( –x ) ( –y ) = | x | | y |

Proof Strategies
Forward Reasoning
is the type of reasoning which starts with the premises and together with axioms, definitions, and known theorems, and then constructs a proof that leads to the conclusion via direct proof.   With proof by contraposition, start with the negation of the conclusion leading to the negation of the premise(s).
Backward Reasoning
is the type of reasoning where proving statement q, find a statement p that can be proven that also has the property p → q.

When confronted with a statement to prove:

  • replace terms by their definitions
  • carefully analyze what the hypotheses and conclusion mean
  • try direct proof which requires forward reasoning
  • if direct proof fails, try a proof by contraposition
  • if none of the above proofs work, try proof by contradiction

MATHEMATICAL INDUCTION


Mathematical Induction
Principle of Mathematical Induction
aims to prove that if propositional function P ( n ) is T, then P ( n + 1 ) is T for all positive integer n = 1, 2, 3, . . ., by completing two steps:

BASIS STEP: Verify that P ( 1 ) is T
INDUCTIVE STEP: Show that the conditional statement P ( n ) → P ( n + 1 ) is T for all positive integers n = 1, ,2, 3, . . .,
i.e. if P ( n ) is T, then P ( n + 1 ) is T

First, prove that P ( 1 ) is T.
Then assume that P ( n ) is T for some arbitrary positive integer n.   P ( n ) is called the inductive hypothesis and MUST be used to prove P ( n + 1).
Then show that under this assumption, P ( n + 1 ) must also be T.

Rule of Inference governing Mathematical Induction
[ P( 1 ) ∧ ∀ k ( P (k) → P ( k + 1 ) ) ] → ∀ n P ( n ), where n = 1, 2, 3, . . .

STRONG INDUCTION AND WELL-ORDERING


Introduction

Strong Induction is very similar to Mathematical Induction in that the basis step of both methods are the same.   However, the inductive step is slightly different.   In Mathematical Induction, if P ( n ) is T, then P ( n + 1 ) must be T.   In Strong Induction, if P ( j ) is T where j does not exceed n, i.e. j = 1, 2, 3, . . ., n, then P ( n + 1 ) is T.


Strong Induction
Principle of Strong Induction
aims to prove that if propositional function P ( j ) is T, then P ( n + 1 ) is T for all positive integer j = 1, 2, 3, . . ., n, by completing two steps:

BASIS STEP: Verify that P ( 1 ) is T
INDUCTIVE STEP: Show that the conditional statement P ( n ) → P ( n + 1 ) is T for all positive integers j = 1, ,2, 3, . . ., n,
i.e. if P ( j ) is T, then P ( n + 1 ) is T

First, prove that P ( 1 ) is T.
Then assume that P ( j ) is T for some arbitrary positive integer n, j = 1, 2, 3, . . ., n.   P ( 1 ), P ( 2 ), P ( 3 ), . . ., P ( n ) are called the inductive hypotheses, that is, that P holds for ALL integers less than or equal to n.
Then show that under this assumption, P ( n + 1 ) must also be T.
P ( 1 ), P ( 2 ), P ( 3 ), . . ., P ( n ) and not just P ( n ) is used to prove that P ( n + 1 ) is T.
